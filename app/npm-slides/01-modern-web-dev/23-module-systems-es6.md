## ES6 模块

* **`import ... from "..."`**: 加载模块并绑定变量
* **`import abc from "..."`**
* **`import { abc } from "..."`**
* **`import { xyz as abc } from "..."`**
* **`import * as abc from "..."`**
* **`import "..."`**
* **`System.import("...")`**
* **`import "./helper.js"`** 和 **`import "jquery"`**
  * 同 commonJs
